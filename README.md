# Books - PHP & MongoDB

## Installation
MongoDB database: `bookdb`. Collection: `books`. Connection string: `mongodb://localhost:27017`. The collection can be imported from `db\books.json`.

The PHP MongoDB driver must be installed:
`composer require mongodb/mongodb`

## PHP MongoDB driver reference
https://www.php.net/manual/en/mongodb.tutorial.library.php

## Tools
MongoDB / PHP8 / Water.css / HTML5
